# Results

This folder contains the Phase 6 Monte Carlo outputs and sensitivity plots.

- `phase6_results.xlsx` - tabulated simulation results.
- `mc_distributions.png` - overview of sampled output distributions.
- `spin_margin_vs_speed.png` - spin friction margin across exit velocity.
- `tornado_tip_ratio.png` and `tornado_slide_margin.png` - stability sensitivity plots.
- Additional tornado plots capture launch force, arrest force, stored energy and contactor speed sensitivities.

The key finding is that stability, especially tip-over and sliding margin, governs the next design phase.
